## Controls
### Movement
* Move Up: Arrow Up
* Move Down: Arrow Down
* Move Left: Arrow Left
* Move Right: Arrow Right

### Inputs
* Number keys to input

### Other
* Undo: u
* Redo: p
* Solve: s
* Check if board is solved: space
* Reset: r
* Quit: q
